# 体重挑战 - 离线版

纯浏览器本地运行的体重挑战管理工具，无需服务器、无需联网、无需审核。

## 特性

- ✅ **完全离线** - 数据存储在浏览器本地，不联网也能用
- ✅ **无需审核** - 不是小程序/App，不需要微信审核或上架
- ✅ **跨平台** - 安卓/iOS/PC 都能用，只要有浏览器
- ✅ **数据分享** - 支持导出/导入 JSON 数据，方便发给朋友
- ✅ **PWA 支持** - 可添加到手机桌面，像原生 App 一样使用

## 快速开始

### 方法 1：直接打开（最简单）

1. 用浏览器打开 `index.html` 文件
2. 开始使用！

### 方法 2：部署到免费服务器（可以分享给朋友）

#### 方案 A：Vercel（推荐）

1. 注册 [Vercel](https://vercel.com)
2. 导入 GitHub 仓库或拖拽上传文件夹
3. 获得一个链接，分享给朋友

#### 方案 B：GitHub Pages

1. 把代码推送到 GitHub 仓库
2. 开启 GitHub Pages 功能
3. 获得 `https://username.github.io/repo` 链接

### 方法 3：添加到手机桌面（最像 App）

#### iOS (Safari)
1. 用 Safari 打开网页
2. 点击底部「分享」按钮
3. 选择「添加到主屏幕」
4. 桌面会出现 App 图标

#### Android (Chrome)
1. 用 Chrome 打开网页
2. 点击菜单「添加到主屏幕」
3. 桌面会出现 App 图标

## 功能说明

### 创建比赛
- 设置比赛名称、开始/结束日期
- 可选设置人数上限（2-100人）
- 支持减重和增重两种模式

### 添加参与者
- 记录姓名、性别
- 设置初始体重、目标体重
- 设置投入金额

### 记录体重
- 选择参与者
- 选择日期
- 输入体重（支持快捷 ±0.5kg）
- 自动判断是否达标

### 排名查看
- 实时排名
- 达标状态显示
- 比赛结束后显示收益

### 数据导出/导入
- 导出为 JSON 文本
- 通过微信/邮件分享给朋友
- 朋友导入后能看到完整数据

## 数据存储

所有数据保存在浏览器 `localStorage` 中：

- **不要清除浏览器数据**，否则数据会丢失
- **可以导出备份**，以防万一
- **不同浏览器数据不互通**

## 技术栈

- 纯 HTML/CSS/JavaScript
- 无框架依赖
- Service Worker 实现离线缓存
- localStorage 本地存储

## 文件结构

```
weight-challenge-h5/
├── index.html          # 入口文件
├── manifest.json       # PWA 配置
├── service-worker.js   # 离线缓存
├── app.js              # 主应用逻辑
├── app.css             # 样式
├── utils/
│   ├── storage.js      # 本地存储封装
│   └── calculator.js   # 计算工具
└── static/
    ├── icon-192.png    # PWA 图标
    └── icon-512.png
```

## 自定义图标

替换 `static/` 目录下的图标文件：
- `icon-192.png` - 192x192 像素
- `icon-512.png` - 512x512 像素

## 注意事项

1. **数据备份**：重要数据建议定期导出备份
2. **浏览器兼容性**：推荐使用 Chrome/Safari/Edge 最新版
3. **隐私模式**：浏览器隐私模式下数据可能无法保存
4. **多设备同步**：目前不支持自动同步，需要手动导出导入
